# AI for Business

This is a Docusaurus test site rendered from Markdown.

## Why this setup

- Markdown files live in `docs/`.
- Docusaurus builds them into a static website.
- The site is configured for GitHub Pages at `/ai-for-business/`.

## Example

```bash
npm run build
```

If this builds successfully, the Markdown rendering path is working.
