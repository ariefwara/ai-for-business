# エージェンティック・エンタープライズ

本シリーズでは、企業がAIツールやコパイロットから**エージェンティック・エンタープライズ**へと移行する方法を解説します。これは、AIエージェントがコンテキストを理解し、エンタープライズシステムを通じてアクションを実行し、人間と協働し、明確なガバナンス管理下に置かれるオペレーティングモデルです。
